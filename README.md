# Embedded Rust with Raspberry Pi Pico 2 W

This repository documents my journey into embedded systems using Rust and the Raspberry Pi Pico 2 W.

Most of my experience has been in software engineering, backend development, and full-stack applications. While those areas focus on building software that runs on top of operating systems and cloud infrastructure, I wanted to gain a deeper understanding of computing from a lower level by learning how software interacts directly with hardware.

To do that, I'm working through the book `impl Rust for RP2350` and completing each project. Along the way, I'll be exploring concepts such as GPIO, PWM, sensors, displays, motors, wireless communication, and embedded programming in Rust.

### Raspberry Pi Pico 2 W Specifications

- RP2350 microcontroller
- Dual-core Arm Cortex-M33 or RISC-V Hazard3 processors
- Up to 150 MHz clock speed
- 520 KB SRAM
- 4 MB onboard flash memory
- 26 GPIO pins
- 4 analog input (ADC) channels
- Wi-Fi and Bluetooth connectivity
- USB 1.1 host/device support
- Programmable I/O (PIO) for custom hardware interfaces
